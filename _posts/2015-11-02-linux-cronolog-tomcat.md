## 安装cronolog

1，<a target=‘_blank’ href=‘http://download.csdn.net/detail/xuhf_1988/9233029’>下载cronolog</a>

2，安装

```sh
cd cronolog-1.6.2/
./configure
sudo make
sudo make install
```

安装完成后，使用`which cronolog`查看

```sh
/usr/local/sbin/cronolog
```

即为安装成功
